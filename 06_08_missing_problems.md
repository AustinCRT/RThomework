what is restful api means
RESTful API stands for Representational State Transfer — it's an architectural style for client-server communication over HTTP. 
Key constraints are: stateless, meaning each request is self-contained; resources identified by URIs like /api/employees/1; and standard HTTP methods — GET, POST, PUT, DELETE. 
An HTTP request has three parts: the URL identifying the resource, headers carrying metadata like Content-Type and Authorization, and the body containing the JSON payload for POST and PUT. 
Finally, we use status codes — 2xx for success, 3xx for redirection, 4xx for client errors, 5xx for server errors.
